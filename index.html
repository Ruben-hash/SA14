BUT Réseaux et Télécommunications : en cours (1ère année)

Le BUT Réseaux et Télécommunications est un diplôme universitaire qui dure 3 années. C'est une formation complète où la théorie se concrétise grâce à la pratique dans de nombreux projets en groupe ou seul :

  • Création d'un site web dynamique avec une base de données SQL
  • Script python pour gérer des données SQL
  • Dimensionnement et créations de réseaux Lan et Vlan intégrant des services réseaux

Deux parcours s'offrent à nous dès la deuxième année : ROM et Cybersécurité. Mon parcours envisagé est la cybersécurité.
